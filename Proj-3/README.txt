How to Run the Project:
1. Install Node.js: Ensure that Node.js (v14.x or higher) and npm (Node Package Manager) are installed on your system.
   You can download it from Node.js official website.

   Required npm Packages: npm install express, npm install ejs, npm install body-parser, npm install express-session, npm install mysql2. 

2. Install MySQL: Ensure MySQL is installed and running on your system.

3. npm install

	3.	Start the Application- node app.js

4.	Access the Application: http://localhost:3000
